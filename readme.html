What's this about then?



The map shows the election result of the 2021 Berlin referendum "Deutsche Wohnen & Co. enteignen" in all Berlin voting districts [1]. A yellow colored district voted in favor of the referendum, a purple colored one voted against it. One peculiarity of the Berlin voting districs is the difference between in-person and mail-in voting districts. Mail-in voting districts may contain more than one in-person voting district. For ease of use all votes within a Briefwahlbezirk (including the in-person votes) are considered. The full data collection and analysis process is lined out here [1]
The report box in the top right corner (or in the bottom section) of your screen provides you with exact numbers of the referendum results within the voting districts that are currently on display, for instance the number of Yes and No votes, the voter turnout and so on. You can select voting district by clicking on them. When pressing CTRL (‚åò on a Mac) while clicking, you add voting districts to your selection. The report box then updates the results.

The options box on the left (or lower) corner of your screen allows to control the map. Per default the map shows a district in a color according to the majority of votes. If you'd like to see the streets of a voting district instead, set a check mark on the transparency option ‚úÖ The same goes in case you'd like to also consider invalid votes in the report box.

The interesting part

You can control Which voting district are shown by filtering according to certain categories. If no filter is active, all voting districts are shown. In case you activate a filter, the report box automatically updates. This way you can find similarities between voting districts. If your on a PC, feel free to use the arrow keys to switch quickly between categories.
